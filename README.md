# Virat Kohli Profile Page (TASK-1)

A clean and simple web profile page dedicated to the legendary Indian cricketer **Virat Kohli**.

## 📌 Project Overview
This project displays a short biography of Virat Kohli, an image, his official Instagram profile link, and an interactive button. It is styled with custom styling to provide a clean and readable layout.

## 📂 File Structure
Here is the layout of the project files:
*   [apex.html](file:///c:/Users/t7907/OneDrive/Desktop/TASK-1/apex.html): The main HTML document containing the structure, layout, biography paragraph, image sources, and interaction script.
*   [taskcolour.css](file:///c:/Users/t7907/OneDrive/Desktop/TASK-1/taskcolour.css): The stylesheet containing typography, custom layouts, colors, and button styles.
*   `insta.png`: Icon image used for the Instagram link anchor.
*   `virat.jpg.webp`: A local webp image asset of Virat Kohli.

## 🚀 Features
*   **Biography Card:** Visual details along with a short paragraph describing Virat Kohli's early life and background.
*   **Instagram Integration:** Clickable Instagram icon pointing to Virat Kohli's official Instagram page.
*   **Interactive Button:** A button styled to match the theme that triggers an alert showing `"This is task-1."` when clicked.
*   **Custom CSS Styling:** Light blue background, centered layouts, black headers, and sleek buttons.

## 🛠️ Technologies Used
*   **HTML5:** Structure and content.
*   **CSS3:** Custom layouts, centering, sizing, and colors.
*   **JavaScript (Embedded):** Interactive button event listener.
